# netcore-docker-rabbit
Sandbox application for .NET Core microservices with RabbitMQ under docker-compose or kubernetes


## Frameworks and components:
1. ASP.NET Core with Blazor UI
2. EntityFramework Core
3. PostgreSQL
4. RabbitMQ
